# dps-gas

Gas mix types and blending models for dive planning.

Part of the [DPS](https://github.com/asakura/dps) project.

## Overview

Calculations for Enriched Air Nitrox (EANx) including:
- Maximum Operating Depth (MOD)
- Equivalent Air Depth (EAD)
- Equivalent Narcotic Depth (END)
- Gas density and toxicity rates (CNS%, OTUs)
- Blending models: Partial Pressure, PSA, and Membrane systems
