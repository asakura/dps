# dps-units

Newtype wrappers for physical units used in dive calculations.

Part of the [DPS](https://github.com/asakura/dps) project.

## Overview

Provides type-safe wrappers for:

- Bar (Pressure)
- Meters (Depth)
- Percent (Fractions)
- Celsius (Temperature)
- And other dive-specific units (CNS%, OTU/min, etc.)
